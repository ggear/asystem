Usage:
  supervisor stats [OPTIONS]

Description:
  Show real-time system statistics

Options:
  -m, --mode string         Method to retrieve stats:
                              'poll'      – Poll system for stats (default)
                              'cache'     – Query stats from cache (if cache is unavailable, fallback to poll mode)
  -f, --format string       Display format based on console width:
                              'auto'      - Uses console width to determine the most spacious format possible (default)
                              'compact'   – For consoles <60 chars wide
                              'relaxed'   – For consoles >120 chars wide
  -p, --poll-period string  Polling interval, uses unit suffixes [s, m, h] (ignored in cache mode, default: "1s")
  -n, --no-stream           Disable continuous streaming, show only the first result and exit
  -h, --help                Display this help message and exit
