Usage:
  supervisor run [OPTIONS]

Description:
  Run the supervisor process to collect and publish system statistics and
  perform supervisory duties.

Options:
  -p, --poll-period duration
      Polling period for caching individual metric raw values,
      with unit suffixes [s, m, h] (default: 1s).

  -b, --bin-period duration
      Bin period for analysing and publishing individual, calculated metric values,
      with unit suffixes [s, m, h] (default: 5s).

  -s, --snapshot-period duration
      Snapshot period for publishing a complete set of calculated metric values,
      with unit suffixes [s, m, h] (default: 5m).

  -h, --help
      Display this help message and exit.
