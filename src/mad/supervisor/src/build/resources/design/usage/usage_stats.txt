Usage:
  supervisor stats [OPTIONS]

Description:
  Show real-time system stats.

Aliases:
  atop
  atops (--mode remote)

Options:
  -m, --mode string
      Stats host selection and retrieval method:
        local
            Show stats from the local host only, directly polled, presented
            as a single cell (default).
        remote
            Show stats from all hosts, read from the supervisor message bus
            and presented as a grid.
        remote[:HOST[,HOST...]]
            Show stats from the specified hosts, read from the supervisor
            message bus and presented as a grid.

  -f, --format string
      Output format based on grid cell width:
        auto
            Select the most spacious format the grid cell allows (default).
        compact
            Optimised for grid cells narrower than 59 columns.
        relaxed
            Optimised for grid cells wider than 100 columns.

  -p, --poll-period duration
      Polling period for caching individual metric raw values,
      with unit suffixes [s, m, h] ignored in remote mode (default: 1s).

  -b, --bin-period duration
      Bin period for analysing and publishing individual, calculated metric values,
      with unit suffixes [s, m, h] ignored in remote mode (default: 5s).

  -r, --refresh-period duration
      Period by which the entire console is cleared and redrawn, with unit
      suffixes [s, m, h], ignored in no stream mode (default: 10m).

  -n, --no-stream
      Disable continuous streaming, display a single console and exits.

  -b, --monochrome
      Disable console colour output.

  -h, --help
      Display this help message and exit.
