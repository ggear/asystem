Usage:
  supervisor [OPTIONS] COMMAND

Description:
  Run supervisor processes.

Commands:
  run
      Run the supervisor process to collect and publish system statistics
      and perform supervisory duties.

  stats
      Show real-time system statistics.

  help
      Display this help message and exit.

Global Options:
  -c --config file
      Path to the configuration file (default: /var/lib/asystem/install/supervisor/latest/image/config.json).

  -v, --version
      Display version information and exit.

  -h, --help
      Display this help message and exit.

Run 'supervisor COMMAND --help' for more information on a command.
